Iconography

Meevo-App utilizes Microsoft Fluent Icons. These icons offer a friendly yet professional geometric aesthetic that perfectly compliments Readex Pro.

Design Directives

  • Weight: Icons should use the 'Regular' weight for standard UI elements.
  • Sizing: Standard sizes are 16px, 20px, and 24px.
  • Color: Icons inherit text color context. They should generally be --color-text-secondary and transition to --color-text-primary on hover/active.
  • Spacing: Always maintain at least 8px gap between an icon and its text label in buttons and lists.

Design Phase

The Design Phase in Meevo-App is a specialized workflow mode focused exclusively on the visual editing of an individual tile (Casilla) or asset.

1. Objetivo

Aislar un elemento complejo (como una Casilla o Carta) para permitir su micro-edición (capas, textos, imágenes, opacidades) sin las distracciones ni la sobrecarga computacional del tablero global.

2. Contexto

Mientras que la fase "Board" sirve para macro-organización (colocar objetos, definir conexiones lógicas), la fase "Design" es un entorno de dibujo vectorial y espacial enfocado en el Asset seleccionado.

3. Concepto

El modelo mental es el de "entrar" en el objeto. Se accede haciendo doble clic sobre el elemento en el Canvas, cambiando instantáneamente las herramientas y atajos disponibles.

4. Visualización

El resto del tablero se oscurece (`opacity-50` o un dim) y los controles de interacción global se bloquean. El elemento enfocado resalta y muestra sus guías de edición (`SnapGuides`, redimensionamiento, etc.).

5. Atajos

  • V: Cursor (Seleccionar/Mover)
  • R: Rectangle (Shape tool)
  • T: Text (Caja de texto)
  • I: Image (Subir/Insertar imagen)
  • Alt + Drag: Duplicar capa (`cursor-duplicate`)
  • Espacio + Drag: Paneo del canvas (`cursor-grabbing`)

6. Importancia de los atajos

Dado que el usuario debe concentrarse en el diseño espacial, forzarlo a mover el mouse hacia una barra de herramientas constantemente rompe su flujo cognitivo. Los atajos garantizan una interacción fluida y profesional (como en Figma o Photoshop).

Cursors

Meevo overrides the default operating system cursors in specific interactive zones using custom PNGs.

Canvas Custom Cursors

cmp-cursor-set
Default
Create
Duplicate
Prohibited
Grab
The application uses custom cursors initialized globally via index.css and toggled dynamically based on state (`activeDesignTool`, `altPressed`, `isPanning`).

Do

Change to 'Grab' (`cursor-grabbing`) instantly when the user holds the Spacebar to pan.

Don't

Don't use `crosshair` for selection tools; `cursor-create` is reserved exclusively for drawing shapes, text, or images.

Tokens & Anatomy

  • Global CSS: Defined with `!important` to force application over browser defaults.
  • Offset: Custom PNG cursors use a `16 16` offset to center the hit area (except default).
  • Classes: `.cursor-create`, `.cursor-duplicate`, `.cursor-prohibited`, `cursor-grabbing`.
